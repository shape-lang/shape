# Shape v0.3 Roadmap

**Generated:** 2026-05-17 post-v0.2.0 ship (main @ 5daea484).
**Status:** Active.

## v0.3 acceptance criteria

- **Operator coverage full set** for user types: arithmetic (Add/Sub/Mul/Div/Mod/Neg) + compound-assign + comparison (Eq/Ord) + bitwise (BitAnd/BitOr/BitXor/Shl/Shr) + Not + Index/IndexMut, all VM == JIT smoke-tested
- **LSP feature-spec en-par with rust-analyzer** for core features: hover, completions, diagnostics, inlay hints, go-to-def, find-references, rename, code-actions, semantic tokens, document/workspace symbols
- **Shape book zero divergence**: every feature in book is in language + every language feature is in book (or has explicit "internal-only" / "deferred to v0.4" cite)
- **Q25.C.6 IC devirtualization landed** (dyn dispatch perf path)
- **v2-raw residuals resolved**: `length_typed_object_empty` + `w17_comptime_*` SIGABRTs
- **Phase 4 imprecision 83 root cause fix**: compiler trait-def return-type propagation gap
- **Smoke matrix 5/5 VM == JIT preserved** at every close
- **0 bad-code merges into main** preserved

## Workstreams

### W1 — Operator coverage (full set)

Mirror Phase 4 Add/AddAssign pattern: trait declaration in `crates/shape-runtime/stdlib-src/core/` + prelude import + UFCS dispatch arm + smoke fixtures VM == JIT.

| Sub-cluster | Operator(s) |
|---|---|
| W1.1 | Sub + SubAssign |
| W1.2 | Mul + MulAssign |
| W1.3 | Div + DivAssign |
| W1.4 | Mod + ModAssign |
| W1.5 | Neg |
| W1.6 | Not |
| W1.7 | Eq (`==`, `!=`) |
| W1.8 | Ord / PartialOrd (`<`, `<=`, `>`, `>=`) |
| W1.9 | BitAnd + BitOr + BitXor + Assign variants |
| W1.10 | Shl + Shr + Assign variants |
| W1.11 | Index + IndexMut |

Dispatch: parallel sub-agents (territory non-overlap — each touches its own stdlib file + per-operator dispatch arm; Phase 4 imprecision 83 fix should land in W6 first to remove return-typeless prelude trait method monomorphization risk). Scope: ~150-300 LoC per sub-cluster.

### W2 — LSP feature-spec en-par with rust-analyzer

**W2.1 Audit-day:** single sub-agent produces `docs/cluster-audits/v0.3-lsp-feature-spec.md` mapping rust-analyzer feature coverage vs `shape-lsp` HEAD. Per-feature gap inventory + per-gap scope estimate + closure-wave partition recommendation.

**W2.2-N Closure-waves:** per audit's per-family partition. Expected families:
- Hover (type info + doc lookup)
- Completions (context-aware; trait methods; named exports; closure params)
- Diagnostics (semantic errors with quick-fix code actions)
- Inlay hints (type hints; param names; closure return types; binding-kind hints; Q25.C.7 cost-class hints folds here)
- Go-to-definition + find-references
- Rename
- Code actions (quick-fixes + refactors)
- Semantic tokens (syntax highlighting)
- Formatting / range formatting
- Document symbols / workspace symbols

Scope: 1 audit-day + 6-10 closure-waves; medium-large workstream.

### W3 — Shape book audit + divergence resolution

**W3.1 Audit-day (5 parallel slice agents; non-overlapping book pages):**

| Slice | Territory |
|---|---|
| A | Language reference (syntax + grammar + types + traits + generics + pattern matching + control flow) |
| B | Stdlib reference (collections + IO + concurrency + temporal + decimal + bigint + string) |
| C | Tutorial + getting-started + installation + first-program |
| D | AI-native + comptime + polyglot interop (Python/TypeScript/C) + annotations |
| E | JIT + perf + sandboxing + permissions + content-addressed bytecode |

Per-slice deliverable: `docs/cluster-audits/v0.3-book-slice-{A..E}-divergence.md` with per-page divergence table + per-divergence disposition (update book / add to language / mark deferred-with-cite).

**W3.2-N Per-slice fix waves:** parallel post-audit per slice's recommendation. Some fixes are doc-only (book update); some are language additions; some are deletion of stale book pages.

Scope: 5 parallel audit agents + 5 parallel fix waves; bounded per-slice.

### W4 — Q25.C.6 IC devirtualization (dyn dispatch perf)

Per `docs/cluster-audits/cluster-2-inventory.md` §I + cluster-1.5 Q25.C close: dyn-dispatch fast-path via inline-cache devirtualization. ~500-1500 LoC LARGE scope; multi-session chain candidate per D-α. Smoke: dyn T dispatch perf regression test + correctness preserved.

### W5 — v2-raw residuals (consolidated)

`length_typed_object_empty` SIGABRT + `w17_comptime_*` SIGABRTs empirical isolation + fix. Single consolidated sub-cluster mirroring `cluster-1.5-v2-raw-empirical-isolation-and-fix` shape (gdb-backtrace empirical isolation; ReceiverGuard or share-accounting fix per root cause).

### W6 — Phase 4 imprecision 83 root cause

Compiler trait-def return-type propagation gap (return-typeless prelude trait method poisons `Vec.map<U>` monomorphization; Phase 4 applied workaround at `add_assign(other: Self): Self` — root cause untouched). Single bounded sub-cluster on compiler side.
